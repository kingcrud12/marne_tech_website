# Charte Graphique - Marne Tech

Ce document regroupe les éléments fondamentaux de l'identité visuelle de l'agence Marne Tech.

## Palette de Couleurs

| Couleur | Hex | Usage |
| :--- | :--- | :--- |
| **Teal Profond** | `#296B7E` | Couleur primaire, Navigation, Titres |
| **Lime Vibrant** | `#B0E096` | Couleur d'accent, Boutons, Highlights |
| **Dark Slate** | `#0f172a` | Pied de page, Fonds sombres |
| **Off-White** | `#F4F6F6` | Fonds de sections, Gris clair |

## Typography

La police de caractère principale est **Inter**. 
- Elle est utilisée pour sa lisibilité exceptionnelle sur écran et son aspect moderne.
- **Titres** : Bold / Black (700-900)
- **Corps de texte** : Regular (400)

## Logo

Le logo officiel de Marne Tech utilise une approche typographique "boxed" :
- **Texte "MARNE"** : En gras, couleur sombre (`#1a3a41`).
- **Texte "TECH"** : En blanc, contenu dans un rectangle aux coins légèrement arrondis avec le fond Teal (`#296B7E`).
- Cette structure symbolise la solidité de l'agence associée à l'innovation technologique encadrée.
