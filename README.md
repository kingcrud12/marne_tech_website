# Marne Tech - Agence Digitale

Site web moderne et premium pour l'agence **Marne Tech**, basée à Paris. Ce projet a été conçu pour offrir une expérience utilisateur fluide et immersive, s'inspirant des standards de design de l'agence Youdemus.

## Fonctionnalités

- **Design Premium** : Esthétique moderne utilisant une palette de couleurs professionnelle (Teal & Lime).
- **Multi-pages** :
  - `index.html` : Accueil et services principaux.
  - `agence.html` : Histoire, valeurs et équipe.
  - `expertise.html` : Détails techniques et FAQ.
  - `realisations.html` : Portfolio complet des projets.
- **Interactivité** :
  - Navigation "Sticky" avec effet de flou (Glassmorphism).
  - Animations de révélation au défilement (Scroll Reveal).
  - Effets de survol dynamiques sur les cartes de services et les boutons.
- **Icônes Professionnelles** : Utilisation de la bibliothèque **Lucide Icons** pour un rendu sobre et moderne.
- **Responsive** : Entièrement optimisé pour mobiles, tablettes et ordinateurs.

## Stack Technique

- **HTML5** : Structure sémantique.
- **CSS3** : Design system personnalisé, Flexbox/Grid, Animations.
- **JavaScript (Vanilla)** : Logique d'animation et de navigation.
- **Lucide Icons** : Bibliothèque d'icônes SVG.

## Structure du Projet

```text
marne_tech_website/
├── assets/
│   ├── identite_visuelle/ # Charte graphique, logo, couleurs
│   └── images/            # Images et fonds d'écran
├── index.html             # Page d'accueil
├── agence.html          # Page "L'Agence"
├── expertise.html       # Page "Expertise"
├── realisations.html    # Page "Réalisations"
├── style.css            # Styles globaux et design system
├── main.js              # Scripts d'animation
└── README.md            # Documentation
```

## Installation

Aucune installation complexe n'est requise. 

1. Clonez ou téléchargez le dépôt.
2. Ouvrez le fichier `index.html` dans votre navigateur préféré.
